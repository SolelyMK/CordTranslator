# CordTranslator
Browser extension created with React to transform coordinates from Geographical format to Rectangular


## HOW TO USE ##
- Download these files into a folder (name it as you wish)
- Then either compress them or laeve them like that
- Go to your browser extension manager
- Switch to a dev mode
- Choose to load packed (compressed folder) or unpacked (basic folder)
- Now it is to your service
